# Modrek

### Getting Started

**These instructions will help you build, release and run the application.**

### Requirements
```bash
  node -v
  ## You should see "v10.x.x"
```

```bash
  yarn -v
  ## You should see "v10.x.x"
```

### How to install

**Clone the repository and install dependencies**

```bash
  git clone https://github.com/910ths/modrek-app.git
  cd modrek-app
  
  # Using yarn
  yarn setup:yarn

  # Using npm
  npm run setup:npm
```
